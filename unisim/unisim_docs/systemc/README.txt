Installing: 
1. Uncompress the beamer-theme-cea-pdf.tar.gz:
		tar zxvf beamer-theme-cea-pdf.tar.gz
2. Go inside the new directory theme-cea
3. In this directory get under root mode:
		sudo bash
3. Copy the "themes" directory to your current beamer (usually at /usr/share/tex/latex/beamer)
		cp -r themes /usr/share/tex/latex/beamer
4. Run the following command:
		texhash
5. That's all, you can leave the root mode if you haven't already done that
