Warning!

Your web browser can`t play the video because none of the installed browser plugins can handle mime type
video/x-msvideo

Tips for playing the videos with your web browser:

  • Internet Explorer on Windows: Windows Media Player 9 or later must be installed. If asked for, allow the browser to run the `Windows Media Player Core` add-on from `Microsoft Corporation`.
  • Firefox on Windows: In Menu Tools > Add-ons > Tab Plugins, Windows Media Player plugin must be enabled, and VLC plugin should be disabled.
  • Firefox on Linux: In Menu Tools > Add-ons > Tab Plugins, Totem plugin must be enabled, VLC plugin should be disabled.
  • Safari on Mac OS X: QuickTime plugin must be enabled

Note: the right video and audio codecs must also be installed on your computer.
