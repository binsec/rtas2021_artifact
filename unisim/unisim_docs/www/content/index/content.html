UNISIM Virtual Platforms

What is it? UNISIM provides several virtual platforms and a framework to ease the development of new virtual platforms...more
Getting it It is open source software. Presently, the UNISIM virtual platforms work on Linux, Windows and Mac OS X host machines, and support the ARM, PowerPC, Star12X, and TMS320C3X target architectures.
Getting support Contact us at supportunisim-vp.com
We can support you in building your own virtual platform, or building it for you, either within an industrial/research project or under a commercial contract.
