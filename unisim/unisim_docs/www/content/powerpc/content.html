PowerPC Virtual Platforms

Overview

The UNISIM PowerPC virtual platforms is a set of research instruction set simulators written in SystemC TLM. Computations on IEEE 754 floating point numbers are emulated using Simfloat++. Altivec instructions are currently decoded but not implemented. These simulators allow debugging the software on a bare desktop PC, without requiring the true hardware, by connecting a GDB client to the simulator through the GDB serial remote protocol. The GDB client can be either the standard text based client (i.e. command gdb), a graphical front-end to GDB (e.g. ddd), or even Eclipse CDT. Each simulators have also a builtin text-based inline debugger.
