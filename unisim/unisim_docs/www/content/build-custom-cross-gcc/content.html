Build a custom GCC cross compiler using crosstool-NG

Prerequisite: crosstool-NG should be preliminarly installed, see tutorial How to build a GCC cross compiler ?.

1. Customize your own gcc cross compiler build

			$ ./ct-ng menuconfig

A user interface allow to customize many things. On exit, at the question "Do you wish to save your new crosstool-NG configuration?", answer "Yes".

#include "images/ct-ng-menuconfig.txt"

#include "images/ct-ng-menuconfig-save.txt"

This will create a file named ".config" within the crosstool-NG directory.

2. Build your custom gcc cross compiler

			$ ./ct-ng build
