UNISIM Virtual Platforms

PowerPC Virtual platforms that include a PowerPC processor going from desktop machines to embedded industrial applications
ARM Virtual platforms of System-on-Chip systems that include an ARM core going from industrial to consummer electronics applications
Star12X Virtual platforms of Star12X based System-on-Chip systems that include an HCS12X core essentially for automotive applications
TMS320C3X Virtual platforms of a TMS320C3X DSP processor essentially for digital signal processing applications

UNISIM Tools

UNISIM GenISSLib An instruction set simulator generator.
UNISIM Toobox An eclipse based tool to easily assemble virtual platforms.
UNISIM Workbench An eclipse based tool to easily use virtual platforms.

Third party tools

GCC cross compiler Some prebuilt GNU GCC cross compilers to compile software for the targets. Cross compilers are essential tools when developping with a virtual platform.
MINGW/MSYS An environment for building native Windows applications with the GNU tool chain. It is an essential tool if you intend to compile the UNISIM virtual platforms under Windows.
