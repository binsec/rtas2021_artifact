UNISIM Virtual Platforms

PowerPC
Virtual platforms that include a PowerPC processor going from desktop machines to embedded industrial applications
ARM
Virtual platforms of System-on-Chip systems that include an ARM core going from industrial to consummer electronics applications
68HCS12X
Virtual platforms of 68HCS12X based System-on-Chip systems that include an 68HCS12X core essentially for automotive applications
TMS320C3X
Virtual platforms of a TMS320C3X DSP processor essentially for digital signal processing applications

UNISIM Tools

GenISSLib
A tool to generate the C++ source of an instruction set simulator from a description of an instruction set architecture (.isa files).
